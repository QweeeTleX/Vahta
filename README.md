# VAHTA — простой сайт с графиком работы 4 бригад

Небольшое приложение, которое позволяет просматривать расписание смен для каждой бригады.  
Также есть возможность **добавлять подработки**, и они отображаются вместе с основным графиком.

---

## 📌 Возможности

- Просмотр графика работы каждой из **4 бригад**  
- Удобное переключение между бригадами  
- Добавление подработок  
- Автоматическое отображение всех смен в общей таблице  
- Простая и интуитивная визуализация расписания  

---

## 🛠️ Технологии

- **HTML**
- **CSS**
- **JavaScript**

---

## 🚀 Как использовать

1. Открыть сайт  
2. Выбрать нужную бригаду  
3. Смотреть график смен по датам  
4. При необходимости — добавить подработку, которая появится в таблице
