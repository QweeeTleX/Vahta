# VAHTA — простой сайт с графиком работы 4 бригад.

Небольшое приложение, где можно посмотреть график смен определенной бригады на установке, также можно добавить подработку и она тоже будет отображаться в общей таблице.

## 📌 Возможности

- Просмотр графика работы каждой из **4 бригад**
- Переключение между бригадами
- Добавление **подработок**
- Отображение подработок вместе с основным расписанием
- Простая и понятная таблица смен

---

## 🛠️ Технологии

- **HTML**
- **CSS**
- **JavaScript (vanilla)**

---

## 🚀 Как использовать

1. Открыть сайт
2. Выбрать нужную бригаду
3. Посмотреть график на конкретные дни
4. При необходимости — добавить подработку, и она появится в расписании
