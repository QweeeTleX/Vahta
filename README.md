# VAHTA — простой сайт с погодой

Небольшое приложение, где можно посмотреть погоду по городу.

## 🚀 Функции
- Поиск города  
- Отображение температуры, ветра, влажности  
- Иконки погоды  
- Быстрый интерфейс

## 🛠️ Технологии
- HTML  
- CSS  
- JavaScript  
- OpenWeather API
